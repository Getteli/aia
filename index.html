Bem vindo a AIA

AIA é uma aplicação desenvolvida usando comando de Voz via JAVASCRIPT.

AIA - Assistente por Inteligência Artificial, mesmo ainda não sendo uma inteligência artificial, o seu futuro ainda será muito promissor e um longo caminho ainda virá.

Ela é gerenciada por um banco de dados. A ideia é algo simples, de forma direta. Seu uso pode ser voltado para totens de atendimento em entreterimento (por exemplo: shoppings, parques, festivais entre outros).

Como seu gerenciamento é por um banco, a ideia é que ela seja usada como um banco de perguntas e respostas. Perguntas como: onde fica o banheiro, explicacoes sobre o quadro X, horario de funcionamento, ajuda, locais e etc.

Por ser uma aplicação via GIT, ela no momento não esta conectada ao banco, mas possui algumas frases de exemplo

Sobre AIA:

Criada em meados de 2018, no inicio da minha trajetoria de programação. Foi atualizada a pouco tempo para página no github, mas sem atualizações relevantes em seu código.

Técnologias usadas:

  • Jquery
  • Web Speech API
  • SQL (para o banco de dados)

comandos exemplos:

  • Olá Aia... / Oi Aia...
  • Onde fica o banheiro ?
  • Onde fica a lanchonete ?
  • Pode me explicar este quadro ?
